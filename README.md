# 前言

欢迎来到本基于web的垃圾分类回收系统项目，此项目是针对计算机专业毕业生设计的实战项目，采用Java语言和MySQL数据库开发。在这里，你将了解到本项目的详细内容、技术选型以及核心代码实现。我们致力于通过这个项目分享，帮助你更好地掌握Java Web开发技能。

## 内容介绍

本项目是一款基于web的垃圾分类回收系统，旨在帮助用户实现便捷的垃圾分类查询、回收及预约服务。系统主要功能包括：垃圾分类查询、预约回收、用户管理、回收员管理等。通过这个项目，我们可以提高人们对垃圾分类回收的意识，促进环保事业的发展。

## 技术介绍

- **语言：** Java
- **使用框架：** Spring Boot
- **前端技术：** JS、Vue、css3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
